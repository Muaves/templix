# Templix

A curated collection of modern, minimal HTML, CSS, and Javascript templates.

## Overview

Templix is built for developers who want to move fast without sacrificing quality.
Each template is designed to be clean, responsive, and easy to extend.

No frameworks. No bloat. Just solid foundations.

## Features

* Clean and minimal design
* Fully responsive layouts (atleast im trying to)
* Zero dependencies
* Easy to customize
* Ready for production use
